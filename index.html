PeaZip free archiver

PeaZip is a free file compression software (zip/unzip tool and rar file extractor) available for MS Windows and Linux systems as portable and installable native software.

The project provides a cross-platform alternative to proprietary software (WinZip, WinRar, etc), and works with over 150+ archive formats and variants:

  • Create 7Z, ARC, BZip2, GZip, PAQ/LPAQ/ZPAQ, PEA, QUAD/BALZ, TAR, UPX, WIM, XZ, ZIP
  • Open and extract: ACE, ARJ, CAB, DEB, DMG, ISO, LHA, RAR, RPM, UDF, ZIPX...

The application deploy a cross-platform front-end GUI for multiple file and archive management Open Source components (7-Zip, FreeArc, PAQ/LPAQ/ZPAQ, PEA, UPX), with special focus on security: support for multiple archive encryption standards for 7Z, ARC, RAR, ZIP, ZIPX file formats (based on AES, Twofish, Serpent algorithms), optional two factor (password + key file) authentication, password manager (optionally encrypted) with random password generator, secure file erase.

The software handle spanned (split) archives (.001, .R01, .Z01, etc) and supports in-archive browsing and searching.

The software is developed using Lazarus IDE (and InnoSetup for Microsoft Windows installers) and released under LGPLv3 license.
