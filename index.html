PeaZip free archiver

PeaZip is a free file compression software (zip/unzip tool and rar file extractor) available for Windows and Linux as portable and installable native software.

The application works with over 150+ archive formats and variants:

  • Create 7Z, ARC, BZip2, GZip, PAQ/LPAQ/ZPAQ, PEA, QUAD/BALZ, TAR, UPX, WIM, XZ, ZIP
  • Open and extract: ACE, ARJ, CAB, DEB, DMG, ISO, LHA, RAR, RPM, UDF, ZIPX...

PeaZip project is aimed to provide a cross-platform front-end GUI for multiple file and archive management Open Source technologies (7-Zip, FreeArc, PAQ, PEA, UPX), with special focus on security: multiple archive encryption standards for 7Z, ARC, RAR, ZIP, ZIPX file (based on AES, Twofish, Serpent), optional two factor authentication, encrypted password manager, secure file delete.

The software handle spanned (split) archives (.001, .R01, .Z01, etc) and supports in-archive browsing and searching.

The program is written in FreePascal using Lazarus IDE, Windows installable packages are scripted with InnoSetup (Delphi compatible syntax) and offers a LGPLv3 alternative to proprietary software (WinZip, WinRar, etc), running as native application on Windows/Win64, Wine/ReactOS, and Linux x86/x86-64, with Linux ARM and BSD ports also available.
